(async) computeDependencyTreeForPackage(name, version) → {Object}

Computes the dependency tree for a given package with a given version. It recursively builds the dependency tree for each dependency until no more dependencies can be found. At the end, it caches the result. If the package was previously cached for the provided version, it is returned without further computation.
