Dependency Tree API

Exposes two APIs for retrieving the dependency tree of an NPM package, with or without a given version.

How to run locally

Run npm start

Call the following APIs at http://localhost:3000:

  • GET /package/<package>, where <package> is the name of a package published to NPM e.g.
curl -X GET http://localhost:3000/package/express
  • POST /package/<package> with JSON body and a provided version field e.g.
curl -X POST http://localhost:3000/package/express -d '{ "version": "latest" }'
  • POST /package/<package> with JSON body and useSemver set to true, so that we pick up the latest version that matches a semver version e.g.
curl -X POST http://localhost:3000/package/express -d '{ "useSemver": true }'

How to test

Run npm t
